# /data/

Synthetic telemetry datasets generated by `/simulator/`.

- `DATASET.md` — written before any model training. Includes simulator
  version/commit, generation date, random seed(s), anomaly taxonomy (with
  ground-truth driving variable per injected anomaly), class balance,
  variable list (cross-referenced to the Variable Registry), and
  train/val/test split definition.
- Generation scripts and raw/processed dataset files.
